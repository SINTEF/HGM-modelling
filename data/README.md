# Size distribution of HGMs

This data is taken from the Supporting Information of Johnson et al., (2022), and is a measured size distribution for Potters Sphericel 25P45. We have extracted the relevant data from the file ```2022EF002883-sup-0004-Data Set SI-S03.xlsx``` and include it here as a csv file, but the data is not otherwise modified.

Johnson, D., Manzara, A., Field, L. A., Chamberlin, D. R., & Sholtz, A. (2022). A controlled experiment of surface albedo modification to reduce ice melt. Earth's Future, 10, e2022EF002883. https://doi.org/10.1029/2022EF002883
